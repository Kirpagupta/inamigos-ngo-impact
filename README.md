# InAmigos Foundation — Impact Site

A single-page website built for **InAmigos Foundation**, a Section 8 registered non-profit based in Bilaspur, Chhattisgarh, working across 28 Indian states on food security, education, women's empowerment, animal welfare, environment, and youth skilling.

🔗 **Official site:** [inamigosfoundation.org.in](https://inamigosfoundation.org.in/)

## About

This site tells the story of InAmigos Foundation's six flagship programmes — Seva, Bachpanshala, Jeev, Udaan, Prakriti, and Vikas — and gives visitors a clear path to volunteer, intern, or partner via CSR.

### Features

- 📖 Story-driven hero, "theory of change" pathway, and programme breakdown
- 📊 Animated impact stats (states reached, interns retained, youth trained)
- 🖼️ Real photo gallery sourced from the foundation's official channels
- 📝 Client-side volunteer application form (opens a pre-filled email — no backend required)
- ❓ FAQ section covering registration status, volunteering, and CSR partnerships
- 🔗 Social links to the foundation's official Facebook, Instagram, and LinkedIn pages
- 📱 Fully responsive, accessible (focus states, reduced-motion support), zero dependencies

## Tech Stack

- Plain **HTML5 / CSS3 / vanilla JavaScript** — no frameworks, no build step
- Google Fonts (Fraunces, Work Sans, IBM Plex Mono)
- Scroll-reveal animations and counters via `IntersectionObserver`

## Getting Started

No build tools needed. Just clone and open the file:

```bash
git clone https://github.com/<your-username>/inamigos-ngo-impact.git
cd inamigos-ngo-impact
open inamigos-ngo-impact.html   # or double-click the file
```

Or serve it locally:

```bash
python3 -m http.server 8000
# visit http://localhost:8000/inamigos-ngo-impact.html
```

## Deploying

This is a static file, so it works out of the box on:
- **GitHub Pages** — enable Pages in repo settings, point it at `main` branch
- **Netlify / Vercel** — drag-and-drop deploy, no config required
